# ConceptGuard XR

**오개념을 설명하는 AI가 아니라, 오개념이 강화되기 전에 막는 AI·XR 과학 튜터**입니다. 학생이 XR 공간에서 회로를 직접 조작하고 자신의 이유를 설명하면, 시스템은 조작 로그·회로 그래프·시뮬레이션 결과·자연어 설명을 함께 분석하여 맞춤형 피드백을 제공합니다.

## 핵심 가치

- 텍스트 답변만 보는 AI 튜터의 한계를 넘어, 학생의 실제 조작 행동까지 분석합니다.
- 직렬/병렬, 전류 보존, 전압 분배, 닫힌 회로 조건 같은 추상 개념을 XR 시각화로 연결합니다.
- 정답을 바로 알려주기보다 힌트, 확인 질문, 반례 시뮬레이션, 직접 설명을 단계적으로 선택합니다.
- 교사는 학생별 반복 오개념, 수정 성공률, 힌트 의존도를 대시보드에서 확인합니다.

## MVP 범위

1. 전기회로 단원: 배터리, 전구, 저항, 스위치, 전선
2. 학습 미션 3종: 닫힌 회로, 직렬/병렬 비교, 저항 변화와 밝기 예측
3. 분석 기능: 회로 그래프 판정, 간단 전류/전압 계산, 오개념 위험 점수화
4. AI 기능: 학생 설명 분석, RAG 기반 힌트 생성, 교사용 세션 요약
5. 평가 기능: 피드백 후 수정 성공률, 오개념 반복률, 과도한 개입 지표

## 빠른 실행 방향

```bash
# 백엔드 예시
cd services/api-fastapi
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Unity 프로젝트는 `apps/xr-client-unity` 아래에 두고, FastAPI 백엔드의 `/analyze` 엔드포인트로 회로 그래프와 학생 설명을 전송합니다.
